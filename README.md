# Patrones de Diseño aplicados a un E-commerce en Java

Este proyecto tiene como objetivo estudiar y documentar patrones de diseño aplicados a un caso práctico: un e-commerce.

La idea no es construir una aplicación completa, sino usar un mismo dominio para entender qué problema resuelve cada patrón, cómo se implementa y cuándo conviene utilizarlo.

## Caso base

El sistema representa un e-commerce simple donde existen:

- Productos
- Clientes
- Pedidos
- Pagos
- Envíos
- Descuentos
- Notificaciones

A partir de este dominio se van a explicar patrones creacionales, estructurales y de comportamiento.

## Categorías de patrones

### Patrones creacionales

Se enfocan en la forma en que se crean los objetos.

### Patrones estructurales

Se enfocan en cómo se organizan y conectan las clases y objetos.

### Patrones de comportamiento

Se enfocan en cómo interactúan los objetos entre sí y cómo se distribuyen responsabilidades.
